Projects

Project 1: Playoff Chance Predictor

Description: This project was a statistics based assignment which I had during my AP CSP class that I took junior year of high school. I chose that I wanted to create a program which would take prior information for the NFL season that year and run simulations in order to determine how likely it was that the Broncos would make the playoffs. The simulations were weighted based on each team's win-loss record up to that point, and the result that I got matched the results of a professional site that I found.
Date completed: Junior Year of High School (2023-2024)
Skills used: Python
